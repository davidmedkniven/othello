Rules

  • Choose a name for your team with care
  • Clone this git repository with git clone https://github.com/davstromb/othello.git
  • Create your own branch with git checkout –b <the name of your team>
  • Create a new Java class in package src/agent/agents and make sure that your class extends the abstract class Agent
  • Don't create any other files than <the name of your team>.java

Hints

  • Each turn is timed to two seconds, unused time is saved for the next turn
  • If your agent throws an exception or if you're not able to return a valid move on time, a random move is choosen for you and all saved time is cleared.
  • Several implemented agents can be found in package src/agent/agents that you can use for inspiration
  • The method newGame is called in the beginning of each game.
  • The method nextMove is called when it is your turn

Score

  • Win = 3p
  • Draw = 1p
  • Loss = 0p
